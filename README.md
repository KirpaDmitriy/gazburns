# Gazburnы

## Общая структура репозитория
- main.py - файл с приложением API
- models.py - файл с моделями используемых данных
- handlers - модуль с ручками
- src - модуль с ML- и вспомогательными утилитами
- database - модуль с моделями и методами для БД

## Стек
- FastAPI - для веб-приложения
- Pydantic - для валидации моделей
- jose - для авторизации пользователя с помощью jwt-токенов
